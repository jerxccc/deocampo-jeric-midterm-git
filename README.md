# Student-Profile

## Midterm Examination Reflection

### 1. What is Git used for?

Git is used to track and save changes in a project.

### 2. What is the difference between Git and GitHub?

Git is a version control tool, while GitHub is a website for storing Git projects.

### 3. Why is branching useful?

Branching lets you work on new features without affecting the main project.

### 4. What command did you use to upload your local commits to GitHub?

I used the `git push origin main` command to upload my project to GitHub.

### 5. What did you learn from this activity?

I learned how to use Git and GitHub to manage and upload my project.
